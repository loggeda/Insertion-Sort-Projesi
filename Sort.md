# Insertion-Sort-Projesi

```
[22,27,16,2,18,6]
[2,22,27,16,18,6] 
[2,6,22,27,16,18] 
[2,6,18,27,16,22] 
[2,6,18,16,27,22]
[2,6,18,16,22,27] 

```

Big-O notation:

```

n(n+1)/2 -> (n^2+n)/2
Big-O : O(n^2)

```

Time comptlexity:
```
Average case: O(n^2) 
Worst case: O(n^2)
Best case: O(n)

```
